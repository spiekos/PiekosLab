Key observations:
• MTBL named % roughly doubles through dedup (plasma: 30.9% → 56.4%; placenta: 49.9% → 68.3%). This is because the unnamed collapse is dominated by the adduct/isotope/fragment graph phase — these are signals from the same molecule detected as [M+H]⁺, [M+Na]⁺, [M+2H]²⁺ etc., which carry no independent annotation and get merged into one representative.
• LIPD stays at 100% named throughout — all features come from LIPID MAPS which has systematic lipid nomenclature; there are no anonymous peaks.
• Annotation errors dominate LIPD dedup (335/537 plasma, 551/843 placenta = ~62–65% of drops). These arise from ambiguous m/z matches where multiple lipid species share a nominal mass, particularly in placental tissue with more complex lipid composition.
• QC filter is the biggest single cut: removes ~89% of raw features in MTBL plasma and ~57–76% in LIPD datasets.

Step notes:
① Raw → Tested: 2 plasma analytes (1 placenta) excluded from differential analysis, likely due to missing values or near-zero variance across all samples.
② Diff analysis (q<0.05, no FC): Mann–Whitney U test with BH correction. This set feeds the model — FC filter intentionally omitted here because log₂(ratio+1) space makes raw FC differences unreliable; LASSO handles effect-size selection downstream.
③ FC filter (|log₂FC| ≥ 0.585, i.e. 1.5× fold change): Applied only for biological reporting. These are the analytes you would cite as "differentially abundant" to your PI. Note TP E has the most biologically significant analytes (15), consistent with its late pregnancy timepoint.
④ LASSO (L1 logistic regression, fit on training split only): Further compresses features by zeroing out weak/redundant predictors. TP E drops 39 → 4 features, suggesting high multicollinearity in the late-timepoint analytes. TP C retains all 68 (LASSO penalty did not eliminate any).

⚠ Critical limitation: Only 18/283 background analytes (6.4%) and 0–3/21–38 hit analytes have KEGG IDs. ORA is statistically underpowered.

Why so few KEGG matches? The MTBL_sop dataset is dominated by xenobiotics, drugs, and environmental contaminants (e.g. Tranexamic Acid, Bupivacaine, Ecgonine, Endothion, pesticide derivatives) and many analytes carry Schymanski Level 2–3 annotations (uncertain structures). These compound classes are largely absent from KEGG's endogenous metabolic pathways.

KEGG-mapped compounds in hits: D-Raffinose (C00492, galactose metabolism) · Pantothenic acid / Vitamin B5 (C00864, pantothenate & CoA biosynthesis) · Acetylhomoserine (C01077, cysteine/methionine metabolism) · Lenticin (C09213, biosynthesis of secondary metabolites). These are interesting endogenous metabolites but too few for robust enrichment.

Recommended alternatives: (1) Chemical class enrichment using ClassyFire/ChemOnt hierarchy via PubChem — much higher coverage for mixed datasets; (2) HMDB API for endogenous metabolite pathway mapping; (3) mummichog for activity metabolomics ORA without needing compound IDs. Dot plots for the tested pathways are available in the Plot Gallery → "Pathway ORA" filter.
