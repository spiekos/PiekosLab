# Outputs

This folder contains the output files generated during the data cleaning and exploratory data analysis phases of the project. The files include filtered compliance logs, statistical summaries, multi-test correlation matrices, and visual timeline reports.

## Directory Contents

### 1. Data Cleaning Outputs (`01_data_cleaning`)

* **`dropped_patients_fitbit_log.txt`**
  * **Source Script:** `preprocess_fitbit.py`
  * **Description:** Contains a table of patient IDs excluded from the Fitbit dataset. It logs the maximum number of consecutive days with missing data (per feature) for each dropped patient.
  * **Exclusion Criteria:** Patients were excluded for a certain metric if their maximum number of consecutive missing days for that metric was **$\ge 7$**.
* **`sum_placental_histo_features.txt`**
  * **Source Script:** `preprocess_placental.py`
  * **Description:** Contains the sum of values for each column corresponding to a placental histopathology feature (calculated after features were encoded to numerical values: 0, 1, 2, 3, etc.). Columns were automatically dropped from the final dataset if the feature was not present in any patient.

### 2. Exploratory Analysis Outputs (`02_exploratory_analysis`)

#### Datasets & Profile Metrics

* **`fitbit_data_analysis.txt`**
* * **Source Script:** `analyze_fitbit.py`
  * **Description:** Comprehensive statistical log of the filtered pregnancy-window Fitbit data. It logs:
    * Total cohort patient size and comprehensive missing values count.
    * Table of completely missing days per patient ID.
    * A metric matrix tracking the maximum consecutive missing day streaks per feature per patient.
    * Total unique calendar dates recorded across the study cohort.
    * Global summary metrics (Median and IQR) for clinical and device features.
    * Trimester-level patient counts tracking active representation.
    * Dense compliance checklists reporting whether each patient achieved an 80%+ valid data density rate across individual tracking features.

#### Visualizations

* **`pregnancy_plots_report.pdf`**
  * **Source Script:** `histogram.py`
  * **Description:** A PDF report containing two comparative histograms:
    1. **All Valid Datapoints:** Plots active patient counts per day across all available valid Fitbit updates, using vertical dotted timeline lines to mark pregnancy trimesters and typical delivery date.
    2. **Pregnancy-Only Datapoints:** An identical plot layout, but strictly truncated to display data captured within the formal pregnancy window.

#### Correlation Analysis Tables

> **Note on Correlation Methodology:** All correlation files below are generated by `correlation.py`. The engine executes a vectorized Spearman rank correlation protocol (**$\rho$**) corrected for multiple comparisons via Benjamini-Hochberg False Discovery Rate (**$FDR \le 0.05$**). Testing is divided into two discrete analysis runs using unique filename prefixes to prevent data collisions:
>
> * **`placenta_` Prefix:** Evaluates the relationship between independent **Placental Histopathology Features** and dependent  **Clinical Delivery Variables** .
> * **`fitbit_` Prefix:** Evaluates the relationship between independent **Trimester-Segmented Fitbit Features** and all  **Clinical & Placental Outcome Targets** .

* **`[prefix]full_correlation_table.txt`**
  * **Description:** The complete, unfiltered statistical matrix containing correlation metrics for every tested pair. Formatted as a scannable Markdown table sorted by independent variable names with redundant labels masked out. Lists **$\rho$**, **$p$**-value, sample size (**$N$**), and computed FDR **$q$**-values.
* **`[prefix]filtered_correlation_table.txt`**
  * **Description:** A filtered subset of the full matrix, retaining only the tested variable pairs that successfully passed the multiple testing significance threshold (**$FDR \le 0.05$**).
* **`[prefix]positively_associated_vars.txt`**
  * **Description:** A flat list containing unique target dependent variables exhibiting significant positive relationships (**$\rho > 0$** and **$FDR \le 0.05$**).
* **`[prefix]negatively_associated_vars.txt`**
  * **Description:** A flat list containing unique target dependent variables exhibiting significant negative relationships (**$\rho < 0$** and **$FDR \le 0.05$**).
