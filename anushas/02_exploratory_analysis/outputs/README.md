# Outputs

This folder contains the output artifacts generated by the data cleaning and exploratory analysis pipeline. The files include quality-control logs, summary tables, correlation results, and figures used for downstream interpretation.

## Main output categories

### 1. Data cleaning logs

These files are written by the preprocessing scripts in the data cleaning stage.

- `dropped_patients_fitbit_log.txt`

  - Source: `01_data_cleaning/preprocess_fitbit.py`
  - Description: Logs the Fitbit metrics for which patients were excluded because they had long consecutive stretches of missing values.
- `sum_placental_histo_features.txt`

  - Source: `01_data_cleaning/preprocess_placental.py`
  - Description: Summarizes the encoded placental histopathology features after cleaning.

### 2. Exploratory analysis summaries

These files summarize the cleaned datasets and the analysis workflow.

- `clinical_data_analysis.txt`

  - Source: `02_exploratory_analysis/analyze_clinical.py`
  - Description: Summary of clinical data quality and structure.
- `fitbit_data_analysis.txt`

  - Source: `02_exploratory_analysis/analyze_fitbit.py`
  - Description: Summary of Fitbit data completeness, pregnancy-window filtering, and trimester-level coverage.

### 3. Correlation analysis outputs

These files are produced by `02_exploratory_analysis/correlation.py`.

- `placenta_full_correlation_table.txt`

  - Full Spearman correlation results for placental histopathology versus delivery-related outcomes.
- `placenta_filtered_correlation_table.txt`

  - The subset of placental correlations that pass the FDR threshold.
- `placenta_positively_associated_vars.txt`

  - Unique outcomes with significant positive placental associations.
- `placenta_negatively_associated_vars.txt`

  - Unique outcomes with significant negative placental associations.
- `fitbit_full_correlation_table.txt`

  - Full Spearman correlation results for Fitbit-derived trimester features versus outcomes.
- `fitbit_filtered_correlation_table.txt`

  - The subset of Fitbit correlations that pass the FDR threshold.
- `fitbit_positively_associated_vars.txt`

  - Unique outcomes with significant positive Fitbit associations.
- `fitbit_negatively_associated_vars.txt`

  - Unique outcomes with significant negative Fitbit associations.
- `demographics_correlation_matrix.csv`

  - Correlation matrix for baseline demographic variables.
- `test3_demographics_vif.txt`

  - Variance inflation factor summary for the demographic collinearity analysis.

### 4. Figures

The figures generated by the analysis scripts are stored in the subfolder `figures/`.

- `figures/demographics_correlation_heatmap.png`

  - Generated by `02_exploratory_analysis/correlation.py`.
  - Heatmap of demographic correlation coefficients.
- `figures/pregnancy_plots_report.pdf`

  - Generated by `02_exploratory_analysis/histogram.py`.
  - PDF report with pregnancy-window Fitbit visualizations.

## Regeneration notes

To regenerate the outputs, run the analysis scripts from the `anushas` project directory:

```bash
python 01_data_cleaning/preprocess_clinical.py
python 01_data_cleaning/preprocess_fitbit.py
python 01_data_cleaning/preprocess_placental.py
python 02_exploratory_analysis/analyze_clinical.py
python 02_exploratory_analysis/analyze_fitbit.py
python 02_exploratory_analysis/correlation.py
python 02_exploratory_analysis/histogram.py
```

The figures are written into the `figures/` subfolder, while the text and table outputs are written directly into this folder.
