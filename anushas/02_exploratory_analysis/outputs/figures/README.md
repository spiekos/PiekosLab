# Figures

This folder contains the main visualization outputs produced during exploratory analysis.

## Files

- `demographics_correlation_heatmap.png`
  - Generated by `02_exploratory_analysis/correlation.py`.
  - Displays a Spearman correlation heatmap for baseline demographic and clinical variables.

- `pregnancy_plots_report.pdf`
  - Generated by `02_exploratory_analysis/histogram.py`.
  - Contains a PDF report with comparative histograms of Fitbit data across all valid datapoints and the pregnancy-only window.

## How to regenerate these figures

From the `anushas` project directory, run:

```bash
python 02_exploratory_analysis/correlation.py
python 02_exploratory_analysis/histogram.py
```

These scripts write their outputs directly into this folder.
